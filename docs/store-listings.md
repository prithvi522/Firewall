## VS Code Marketplace Listing

### Title

AI Prompt Firewall

### Short Description

Scan the active editor, selected text, or pasted content locally for prompt injection signals.

### Long Description

AI Prompt Firewall helps you inspect text for prompt injection, hidden instructions, encoded payloads, and unsafe content before it reaches an AI workflow. It auto-starts a local backend on first run.

Use it to:

- Scan the active file in VS Code.
- Scan selected text or pasted content in the webview scanner.
- Review risk scores and suspicious content indicators.
- Auto-create a local Python environment and start the bundled backend on first run.

The extension is designed for developers, reviewers, and security-minded teams that want a quick way to inspect AI input before it is used in production or in a prototype.

### Feature Bullets

- Scans active editor content directly from VS Code.
- Supports pasted text and selected text in the scanner view.
- Highlights prompt-injection style attacks and encoded payloads.
- Auto-installs backend dependencies into a local virtual environment.
- Provides a simple review flow for suspicious content.

## Chrome Web Store / Edge Add-ons Listing

### Title

AI Prompt Firewall

### Short Description

Scan the current page, pasted text, or uploaded files locally for prompt injection signals.

### Long Description

AI Prompt Firewall is a lightweight security tool for inspecting AI input before it reaches a model or agent. It scans page text, pasted text, and uploaded files locally to detect suspicious instructions, hidden payloads, and harmful prompt-injection patterns.

Use it to:

- Scan the current web page with one click.
- Scan pasted text without leaving the popup.
- Upload a file and inspect it locally.
- Review detailed risk results before sharing content with an AI system.

This extension is intended for users who want an extra layer of safety when working with AI tools, internal documents, or agent workflows.

### Feature Bullets

- One-click page text scanning.
- Paste text directly into the popup for analysis.
- Upload local files for inspection.
- No backend setup required.
- Privacy policy included for store review.
