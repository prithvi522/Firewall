AI Prompt Firewall — Privacy Policy

Last updated: 2026-05-23

This privacy policy describes how the AI Prompt Firewall browser extension ("the Extension") collects, uses, and shares data when you use it.

1. Data Collected

  • Page text: When you explicitly click "Scan page text", the extension captures the visible page text (document body) and sends it to the configured backend API for analysis. The extension does not automatically or silently send page text.
  • Uploaded files: If you upload a file via the extension popup, that file is sent to the configured backend for scanning.
  • API URL: The extension stores the API URL in-memory in the popup only; it is not sent to any server by the extension itself.

2. Purpose

The extension sends page text or uploaded files to the configured backend for safety analysis (threat detection, abusive language, sentiment, etc.).

3. Third-party sharing

The extension itself does not forward data to third parties. If you configure a remote backend, that backend operator may process or store data according to their own policies — review the backend's privacy policy before using a hosted service.

4. Retention

The extension does not retain scan data locally beyond displaying the result in the popup. Any retention is controlled by the backend service to which you send data.

5. Security

You are responsible for selecting a trusted backend. When using a remote API, use HTTPS and authentication.

6. User controls

  • Only scan when you explicitly request it.
  • Do not configure the extension to point at untrusted backends.

7. Contact

For questions about this privacy notice, contact the project maintainer at: maintainer@example.com

This is a template privacy policy for demonstration. Review legal requirements (GDPR, CCPA) before distributing publicly and update contact information accordingly.
