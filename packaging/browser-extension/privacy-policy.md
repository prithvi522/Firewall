AI Prompt Firewall — Browser Extension Privacy Policy

Last updated: 2026-05-23

This privacy policy describes how the AI Prompt Firewall browser extension ("the Extension") collects, uses, and shares data when you use it.

1. Data Collected
- Page text: When you explicitly click "Scan current page", the extension captures the visible page text (document body) and analyzes it locally in the browser.
- Uploaded files: If you upload a file via the extension popup, that file is analyzed locally in the browser.
- Pasted text: Text you enter in the popup is analyzed locally in the browser.

2. Purpose
- The extension analyzes text and files locally for safety signals such as prompt injection patterns, suspicious instructions, and unsafe content.

3. Third-party sharing
- The extension itself does not forward data to third parties.

4. Retention
- The extension does not retain scan data locally beyond displaying the result in the popup.

5. Security
- Because analysis happens locally, no backend or network authentication is required for normal use.

6. User controls
- Only scan when you explicitly request it.
- Only scan content you intend to review.

7. Contact
- For questions about this privacy notice, contact the project maintainer at: Ps2domain@gmail.com


This is a template privacy policy for demonstration. Review legal requirements (GDPR, CCPA) before distributing publicly and update contact information accordingly.
